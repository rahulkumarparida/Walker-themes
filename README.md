# Walker Themes

These are custom Walker layouts generated by me only for omarchy. Each theme lives in its own folder, named after the theme, and includes its own `style.css` plus preview images.

Each install snippet is in a code block so you can use the copy button in the top-right when viewing this README on GitHub.

## 01_blackglass
Custom Walker layout with a black glass look and a sleek dark presentation.

Copy command:

```sh
git clone https://github.com/rahulkumarparida/Walker-themes.git  /tmp/walker-repo && cp  /tmp/walker-repo/01_blackglass/style.css  ~/.local/share/omarchy/default/walker/themes/omarchy-default/style.css  && rm -rf  /tmp/walker-repo&& omarchy-restart-walker
```

## Conclusion
The goal of this collection is to keep a ready-made set of already configured Walker layouts in one place. I generated these themes so it is easier to explore different visual styles, switch between them quickly, and use Walker without needing to set up each layout from scratch every time.
